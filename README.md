# Производственная практика "Разработка и поддержка программных модулей"

## Что находится в репозитории

1. order_module.py - Модуль который разрабатывался в течении практики
2. test_order_module.py - Файл тестирования для модуля
3. Отчет.docx - Отчет о проведенной практики с описание 

## Как запустить модуль и протестировать его

### Запуск

```Bash
python order_module.py # Модуль 
```

### Тесты

```Bash
python -m unittest test_order_module.py # Обычный прогон

python -m unittest -v test_order_module.py # Подробный тест
```
