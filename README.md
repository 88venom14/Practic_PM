# Производственная практка "Разработка и поддержка программных модулей"

## Что находить в репозитории

1. order_module.py - Модуль который разрабатывался в течении практики
2. test_order_module.py - Файлы тестирования для модуля
3. Отчет.docx - Отчет о проведенной практики с описание работы кода
4. requirements.txt - Файл с зависимостями

## Как запустить модуль и протестировать его

### Запуск

```Python
python order_module.py # Модуль 
```

```Python
python -m unittest test_order_module.py # Обычный прогон

python -m unittest -v test_order_module.py # Подробный тест
```
